5. Обновить `onboarding.md` — если изменились приоритеты

**ИИ обновляет этот файл САМ, без вопросов, после каждой задачи:**
- Какие баги исправлены
- Какие файлы добавлены/удалены
- Что перестало быть актуальным

**ИИ спрашивает пользователя ТОЛЬКО если:**
