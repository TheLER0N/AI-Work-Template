# 🤖 AI Work — Универсальный шаблон AI-файлов для любого проекта

## Что это?

Это **готовый набор из 13 AI-файлов**, который можно подключить к ЛЮБОМУ проекту.
Скопировал → попросил ИИ настроить → ИИ заполняет реальной информацией → работаешь.

## Быстрый старт

### 1. Скопируй папку AI в свой проект

**Windows:**
```cmd
xcopy AI_work "C:\Путь\К\Проекту\AI" /E /I /Y
```

**Linux/Mac:**
```bash
cp -r AI_work /путь/к/проекту/AI
```

### 2. Скажи ИИ: "Настрой AI-файлы под этот проект"

ИИ сам:
- ✅ Просканирует структуру проекта
- ✅ Заполнит `architecture.md` — описание компонентов
- ✅ Заполнит `design.md` — цвета, стили, UI
- ✅ Заполнит `task.md` — статус и задачи
- ✅ Заполнит `tasks.md` — баги и TODO
- ✅ Заполнит `security.md` — уязвимости и защита
- ✅ Заполнит `roadmap.md` — план развития
- ✅ Заполнит `ideas.md` — идеи для фич с ID
- ✅ Заполнит `onboarding.md` — быстрый ввод в контекст
- ✅ Заполнит `rules.md` — правила работы с проектом
- ✅ Заполнит `workflow.md` — процесс разработки
- ✅ Заполнит `user-responses.md` — ответы пользователя
- ✅ Заполнит `changelog.md` — история изменений
- ✅ Заполнит `bug-handling.md` — правила обработки багов

### 3. Начни работать

Теперь в каждой сессии ИИ будет читать эти файлы и **сразу знать контекст**.

## 📁 Структура файлов

| Файл | Назначение | Когда обновляется |
|------|-----------|-------------------|
| `onboarding.md` | Быстрый ввод в курс дела (читать первым) | При смене приоритетов |
| `task.md` | Текущая задача от пользователя + связь с ideas.md | При каждой новой задаче |
| `tasks.md` | Детализированный список багов и задач + связь с ideas.md | После каждого изменения |
| `architecture.md` | Полная архитектура проекта | При изменении структуры |
| `rules.md` | Правила работы ИИ с проектом | При новых правилах |
| `user-responses.md` | Ответы пользователя на вопросы | После каждого ответа |
| `design.md` | Дизайн-система (UI/UX, цвета, стили) | При изменении UI |
| `security.md` | Политика безопасности | при изменении защиты |
| `workflow.md` | Рабочий процесс (разбиение на подпланы) | При изменении процесса |
| `roadmap.md` | Дорожная карта проекта | При добавлении фаз |
| `ideas.md` | Банк идей с ID для связи с task.md | При новых идеях |
| `changelog.md` | История изменений с кодом ДО/ПОСЛЕ | После каждого изменения |
| `bug-handling.md` | Правила обработки багов | При новых типах багов |

## 🔄 Как работает связь ideas → tasks → task

```
ideas.md (БАНК ИДЕЙ — все возможные фичи с ID)
    ↓ (пользователь выбирает ID)
tasks.md (TODO — запланировано к реализации)
    ↓ (пользователь пишет в task.md "сделай [ID]")
task.md (ТЕКУЩАЯ ЗАДАЧА — что делается сейчас)
    ↓ (выполнено)
changelog.md (ИСТОРИЯ — что и когда сделано)
```

**Пример:**
1. Пользователь пишет в `task.md`: "Сделай `FEAT-001`"
2. ИИ читает `ideas.md` → находит `FEAT-001` → понимает что нужно сделать
3. ИИ создаёт подпланы в `tasks.md` → выполняет
4. Обновляет `changelog.md` → история изменений

## 🚀 Автоматизация (ai-automator.ps1)

В комплекте идёт скрипт для автоматизации:

```powershell
# Просканировать проект и заполнить AI-файлы
.\ai-automator.ps1 scan "C:\Путь\К\Проекту"

# Показать статус всех AI-файлов
.\ai-automator.ps1 status

# Проверить целостность AI-файлов
.\ai-automator.ps1 check

# Предложить следующую задачу
.\ai-automator.ps1 suggest

# Показать идею по ID
.\ai-automator.ps1 link FEAT-001

# Git commit
.\ai-automator.ps1 commit "Описание изменений"

# Создать бэкап AI-файлов
.\ai-automator.ps1 backup
```

## 💡 Советы по использованию

### Для новых проектов
1. Скопируй папку AI
2. Попроси ИИ: "Настрой AI-файлы под этот проект"
3. ИИ просканирует проект и заполнит файлы
4. Проверь что ИИ заполнил правильно
5. Начни давать задачи через `task.md`

### Для существующих проектов
1. Скопируй папку AI
2. Заполни `task.md` — что сейчас актуально
3. Заполни `tasks.md` — известные баги и задачи
4. Остальные файлы заполнит ИИ

### В новых сессиях ИИ
1. ИИ автоматически читает все AI-файлы
2. Сразу знает контекст проекта
3. Может продолжить с места где остановился

## 🆘 Troubleshooting

### ИИ не читает AI-файлы
- Убедись что папка называется `AI` (не `ai` или `Ai`)
- Проверь что все 13 файлов на месте
- Напомни ИИ: "Прочитай все AI-файлы перед началом"

### Файлы устарели
- Попроси ИИ: "Перечитай все AI-файлы и обнови"
- Или удали папку AI и скопируй заново

### Конфликт между файлами
- Попроси ИИ: "Проверь AI-файлы на противоречия"
- ИИ обязан делать это каждые 3 задачи
